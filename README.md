# Erdos90

This repository contains a formal Lean proof of OpenAI's 2026
counterexample to the Erdős unit distance conjecture.  See [this
Leanprover Zulip
thread](https://leanprover.zulipchat.com/#narrow/channel/583339-AI-authored-projects/topic/Erd.C5.91s.20unit-distance.20counterexample.2C.20exponent.20above.201/near/606778508))
for more information.

At the moment, the repository contains two directories:
`src/original/` is the original proof from the model, while
`src/submission/` is (essentially) the submission provided to
[lean-eval](https://lean-lang.org/eval/problems/erdos_unit_distance_conjecture_false/).
